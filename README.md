## 🚀 Getting Started

### Launching the Frontend Visualizer
The web portal is built as an ultra-lightweight, self-contained single-file client utilizing procedural generation via HTML5 Canvas and Tailwind CSS. It is also hosted online on the official web portal at [http://www.theomnipresentstream.com/](http://www.theomnipresentstream.com/).

1. Navigate to the `web_portal/` directory.
2. Open `index.html` in any modern web browser supporting the Web Audio API.
3. To activate the live Y3 Upper Administration Console, paste your API credentials into the secure local sandbox panel. (All keys are strictly contained inside your browser's `localStorage`).

### Running the Backend Engine
The Python simulator evaluates the underlying complex algebraic formulas and telemetry loops.

1. Ensure you have Python 3.8+ deployed in your environment.
2. Run the master runtime thread:

```bash
python3 core_engine/stream_physics.py
```
